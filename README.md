## üöÄ The Vision

Everyone has hobbies they excel at. They have thousands of hours of practice in those hobbies. Let's make use of that existing knowledge through transfer learning and visual metaphors, using **Gemini 3** and **Nano Banana Pro**.

**Our Core Promise:**
- **Translation, Not Just Explanation:** We don't just dumb things down. We map the *structural logic* of a concept (e.g., Transformer Attention) to a domain you master (e.g., a Quarterback scanning receivers).
- **Visual Metaphor Learning:** Text isn't enough. We generate custom visual metaphors that mirror your mental model.
- **Active Recall:** We verify understanding through gamified quizzes that can be answered by inspecting the generated image.
- **Context Switch:** If you don't understand the first explanation, we can try a different visual language, a different metaphor, and/or a different persona.

## üèóÔ∏è Architecture & Tech Stack

This project is a showcase of the latest in the Gemini 3 family of generative AI.

### **The Intelligence Layer**
- **Gemini 3:** The brain of our operation. We use Gemini 3's advanced reasoning capabilities to build the "Metaphor Bridge" - the logic layer that connects source domains (gaming, sports) to target domains (CS, physics).
- **Nano Banana Pro:** Our visual engine. We leverage Nano Banana Pro's ultra-fast image generation models to create precise, metaphor-aligned diagrams in real-time.

### **The Application Layer**
- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) 
- **State:** React Hooks
- **Language:** TypeScript
